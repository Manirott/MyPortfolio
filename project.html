Bike Wheeling Detection System

IoT Twilio

A safety device utilizing the MPU6050 sensor to detect bike wheelies. When dangerous tilts are detected, it triggers a buzzer and sends SMS alerts via Twilio to notify concerned parties.

Load based Speed Control System

IoT

Developed an intelligent motor control system where the speed adjusts dynamically based on the applied load. This system enhances energy efficiency and performance in environments like conveyor belts or smart manufacturing. A load cell detects weight in real-time and sends data to a microcontroller, which then regulates the motor speed accordingly.
